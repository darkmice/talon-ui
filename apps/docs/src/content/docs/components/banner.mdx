## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `tone` | `'info' \| 'success' \| 'warning' \| 'error'` | `'info'` | Background tone + left-bar colour + default icon. |
| `title` | `ReactNode` | — | Bold headline. |
| `children` | `ReactNode` | — | Body text. |
| `icon` | `ReactNode` | tone default | Override the default tone icon. |
| `action` | `ReactNode` | — | Right-aligned action element (typically a `Button`). |
| `onDismiss` | `() => void` | — | When set, render the X dismiss button. |

## Don't

- Don't use `Banner` for transient confirmations (use `Toast`).
- Don't stack more than 2 Banners — collapse them into a single composite or convert to a notification list.
