## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `'sm' \| 'md'` | `'md'` | 28×16 / 32×18 px track. |
| `tone` | `'primary' \| 'danger'` | `'primary'` | On-state background. |
| `checked` | `boolean` | — | Controlled state. |
| `defaultChecked` | `boolean` | — | Uncontrolled initial state. |
| `onCheckedChange` | `(checked: boolean) => void` | — | Change handler. |
| `disabled` | `boolean` | `false` | Disables the control. |

## Don't

- Don't use `Switch` for choosing between multiple options — use `RadioGroup`.
- Don't combine with destructive labels without explicit confirmation. Toggles imply instant effect.
