## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Heights 28 / 36 / 44. |
| `tone` | `'default' \| 'invalid'` | `'default'` | `invalid` paints the border red. Auto-derived from `aria-invalid`. |
| `prefix` | `ReactNode` | — | Slot rendered before the input. |
| `suffix` | `ReactNode` | — | Slot rendered after the input. |
| `wrapperClassName` | `string` | — | Class applied to the outer `<label>`. |
| `className` | `string` | — | Class applied to the inner `<input>`. |

All native `<input>` attributes are accepted (except `size`, which is shadowed by the visual `size` variant).

## Don't

- Don't put `<Input>` inside `<Input>` — the label wrapper assumes a single inner input.
- Don't fight the focus ring; the wrapper supplies it.
