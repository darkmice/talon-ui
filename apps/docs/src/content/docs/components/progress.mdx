## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number` | required | Current value (0..max). |
| `max` | `number` | `100` | Maximum value. |
| `type` | `'linear' \| 'circle'` | `'linear'` | Visual layout. |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Linear: h-1/1.5/2; circle: 32/56/80 px. |
| `status` | `'normal' \| 'success' \| 'warning' \| 'error'` | `'normal'` | Fill colour. |
| `showInfo` | `boolean` | `true` | Render the percentage label. |
| `format` | `(value, max) => ReactNode` | percentage | Custom label renderer. |
| `strokeWidth` | `number` | size-derived | Circle stroke width. |
| `label` | `string` | `'Progress'` | `aria-label`. |

## Don't

- Don't show Progress without a known endpoint — use Spin for indeterminate states.
- Don't combine Progress with a separate "%" suffix; the component already labels.
