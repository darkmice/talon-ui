## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `status` | `'success' \| 'error' \| 'warning' \| 'info' \| '403' \| '404' \| '500'` | `'info'` | Sets icon + colour + default title. |
| `icon` | `ReactNode` | status default | Override the default status icon. |
| `title` | `ReactNode` | status default | Headline (`text-h2`). |
| `description` | `ReactNode` | — | Body copy. |
| `extra` | `ReactNode` | — | CTA row (typically one or two `Button`s). |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Icon + vertical padding scale. |

## Don't

- Don't use Result for transient confirmations (use Toast).
- Don't put multiple Results on a page — it's a terminal state panel.
