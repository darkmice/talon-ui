## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number` | — | Controlled value. |
| `defaultValue` | `number` | `0` | Uncontrolled initial value. |
| `count` | `number` | `5` | Number of stars. |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Star pixel size 14 / 16 / 20. |
| `allowHalf` | `boolean` | `false` | Enable 0.5 step. |
| `readOnly` | `boolean` | `false` | Display-only. |
| `disabled` | `boolean` | `false` | Display-only + greyed out. |
| `onValueChange` | `(value: number) => void` | — | Commit handler. |
| `label` | `string` | `'Rating'` | `aria-label`. |

## Don't

- Don't ship rating without a textual readout — the star value alone is hard to read at a glance.
- Don't display values like 3.7 when `allowHalf` is off. Round on display or enable halves.
