## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `steps` | `StepperStep[]` | required | Array of `{ title, description?, status?, icon? }`. |
| `current` | `number` | required | 0-indexed current step. |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction. |
| `size` | `'sm' \| 'md'` | `'md'` | Indicator circle size (24 / 28 px). |
| `onStepClick` | `(index: number) => void` | — | When set, indicators become buttons for jump navigation. |

## Don't

- Don't ship a Stepper without explicit progress; users need to know where they are.
- Don't use Stepper as a tab strip — that's `Tabs`.
