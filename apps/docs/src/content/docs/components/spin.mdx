## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Spinner icon size (14 / 20 / 28 px). |
| `tone` | `'primary' \| 'inverse'` | `'primary'` | Spinner colour. |
| `tip` | `ReactNode` | — | Caption rendered below the spinner. |
| `spinning` | `boolean` | `true` | Toggle the overlay (only meaningful when wrapping children). |
| `delay` | `number` | `0` | Wait this many ms before showing the overlay (avoids flashing on fast requests). |
| `children` | `ReactNode` | — | When provided, renders the spinner as an overlay on top of the children. |

## Don't

- Don't use Spin for non-async loading (use Skeleton for content placeholders).
- Don't ship Spin without an outer container width — overlay needs measurable bounds.
