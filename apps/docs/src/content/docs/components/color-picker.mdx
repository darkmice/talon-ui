## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `string` | — | Controlled hex (`'#RRGGBB'`). |
| `defaultValue` | `string` | `'#4F60FF'` | Uncontrolled initial value. |
| `onValueChange` | `(value: string) => void` | — | Commit handler. |
| `presets` | `string[]` | Talon brand + utility palette | Preset grid colours. |
| `showHexInput` | `boolean` | `true` | Render the hex input row. |
| `showSystemPicker` | `boolean` | `true` | Render the native `<input type="color">` swatch. |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Trigger height. |
| `tone` | `'default' \| 'invalid'` | `'default'` | Border treatment. |
| `disabled` | `boolean` | `false` | Disables the trigger. |
| `label` | `string` | `'Pick a color'` | `aria-label`. |

## Don't

- Don't ship a giant preset list — keep it under ~18 for a compact picker.
- Don't expect alpha support in Phase 1. Hex is `#RRGGBB` only.
