## Props

### Tabs (Radix Root)
| Prop | Type | Description |
|---|---|---|
| `value` | `string` | Controlled active value. |
| `defaultValue` | `string` | Uncontrolled initial active value. |
| `onValueChange` | `(value: string) => void` | Change handler. |
| `orientation` | `'horizontal' \| 'vertical'` | Default `horizontal`. |

### TabsList / TabsTrigger
| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Font scale; affects only typography (the underline indicator stays 2px). |

`TabsTrigger` accepts a `disabled` prop.

## Don't

- Don't ship more than ~6 tabs — overflow makes scanning hard. Use a sidebar or breadcrumb instead.
- Don't nest Tabs more than 1 level deep — confusing focus model.
