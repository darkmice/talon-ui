## Exports

`Collapse` (Radix Root — accepts `type: 'single' | 'multiple'`, `value`, `defaultValue`, `onValueChange`, `collapsible`), `CollapsePanel`, `CollapseHeader` (accepts `extra` ReactNode), `CollapseContent`.

## Don't

- Don't put long-form prose inside Collapse — readers skim, and hidden long text gets missed.
- Don't nest Collapse more than 2 levels deep — focus model gets confusing.
