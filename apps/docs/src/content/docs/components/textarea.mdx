## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Min-heights 28 / 36 / 44. |
| `tone` | `'default' \| 'invalid'` | `'default'` | `invalid` paints the border red. |
| `autosize` | `boolean` | `false` | Match height to content via a layout effect. |
| `wrapperClassName` | `string` | ‚Äî | Class applied to the outer `<label>`. |
| `className` | `string` | ‚Äî | Class applied to the inner `<textarea>`. |

All native `<textarea>` attributes are accepted (except `size`).
