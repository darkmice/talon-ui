## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `'sm' \| 'md'` | `'md'` | 14 / 16 px. |
| `tone` | `'primary' \| 'danger'` | `'primary'` | Background colour when checked. |
| `checked` | `boolean \| 'indeterminate'` | — | Controlled state. |
| `defaultChecked` | `boolean` | — | Uncontrolled initial state. |
| `onCheckedChange` | `(checked: boolean \| 'indeterminate') => void` | — | Change handler. |
| `disabled` | `boolean` | `false` | Disables the control. |

## Don't

- Don't surround `<Checkbox>` with two clickable wrappers — `<label>` is enough.
- Don't use indeterminate as a permanent state. It signals "partial" and should resolve.
