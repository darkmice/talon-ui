`BusinessRows` ships **four** Talon-Pilot-specific row helpers — designed to be dropped into the product's settings panels, runtime dashboards, and risk reports. They aren't generic Antd parts; they're the named, codified rows from the design system.

## FileRefRow (design.md §6.20)

Path + timestamp; truncates the middle of the path; hovers indigo-subtle; can be wrapped in a link.

## RoleRow (design.md §6.25)

Role icon + name (+ optional manage link) + member count + AvatarGroup.

## RuntimeRow (design.md §6.28)

Runtime device with status, engine chip, leases, and stacked CPU/MEM mini progress bars. Offline rows dim to 70%.

## RiskRow (design.md §6.31)

Amber alert icon + message + go-fix action (link or button).

## Don't

- Don't use these rows outside their named context — they're Talon-Pilot specific patterns, not generic table rows.
- Don't ship `RuntimeRow` without `status` — that's the load-bearing prop.
