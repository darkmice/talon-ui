`Space` arranges children with consistent token-based gaps. Use it instead of ad-hoc `flex` + `gap-*` utilities when the spacing is part of a documented surface.

## Sizes

| Token | Pixels |
|---|---|
| `xs` | 4 |
| `sm` | 8 |
| `md` (default) | 12 |
| `lg` | 16 |
| `xl` | 20 |

## Don't

- Don't use `Space` for grid layouts — use CSS grid directly when the layout has 2D structure.
- Don't pass arbitrary inline `style={{ gap: ... }}` — go through the `size` prop so the gap stays tokenised.
