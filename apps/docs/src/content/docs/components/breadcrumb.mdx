## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `items` | `BreadcrumbItem[]` | required | Path items. Last item is the current page. |
| `maxItems` | `number` | `0` | When > 0 and items exceed this, middle items collapse. |
| `separator` | `ReactNode` | `'/'` | Separator between items. |
| `size` | `'sm' \| 'md'` | `'sm'` | `sm` = 12 / `md` = 14 typography. |
| `label` | `string` | `'Breadcrumb'` | `aria-label` for the nav element. |

## Don't

- Don't render breadcrumbs for two-level paths — overkill.
- Don't make the last item clickable; it's the current page.
