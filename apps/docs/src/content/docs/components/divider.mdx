`Divider` is the standard 1px separator. Horizontal renders an `<hr>`; vertical renders an inline `<span>` you can drop between flex children.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Direction. |
| `tone` | `'default' \| 'strong' \| 'subtle'` | `'default'` | Border-color token. |
| `children` | `ReactNode` | — | Label rendered between two thin rules (horizontal only). |

`role="separator"` and the appropriate `aria-orientation` are always set.

## Don't

- Don't stack three or more horizontal dividers in a row — increase whitespace instead.
- Don't put a vertical divider inside a flow-text paragraph; use it only inside flex / inline-flex toolbars.
