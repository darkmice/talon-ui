## Props

### Descriptions

| Prop | Type | Default | Description |
|---|---|---|---|
| `columns` | `number` | `1` | Grid column count. |
| `size` | `'sm' \| 'md'` | `'md'` | Row min-height (28 / 32). |
| `layout` | `'horizontal' \| 'vertical'` | `'horizontal'` | Label-value layout per row. |

### DescriptionsItem

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `ReactNode` | required | Caption-coloured label. |
| `children` | `ReactNode` | required | Value contents (text, Tag, link, etc.). |
| `mono` | `boolean` | `false` | Render value in monospace (use for IDs, paths, timestamps). |
| `span` | `number` | `1` | Number of columns this item spans. |

## Don't

- Don't pair Descriptions with `<table>` for tabular data — use a real Table component.
- Don't nest Descriptions inside Descriptions; flatten the data instead.
