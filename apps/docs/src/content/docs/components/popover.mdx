`Popover` is a persistent overlay anchored to its trigger. Unlike `Tooltip` it requires explicit click to open and stays open until the user dismisses it.

## Exports

| Name | Description |
|---|---|
| `Popover` | Root state container. |
| `PopoverTrigger` | The element that toggles the popover. |
| `PopoverAnchor` | Optional positioning anchor when the trigger and visual anchor differ. |
| `PopoverContent` | The floating surface. Accepts `width` (`'sm' \| 'md' \| 'lg' \| 'xl'`). |
| `PopoverClose` | Optional X close button absolutely positioned in the top-right. |
| `PopoverArrow` | Optional visual arrow pointing back to the trigger / anchor. |

## Don't

- Don't put long-form forms inside `Popover` — use `Modal` or `Drawer` for that.
- Don't nest two popovers; combine into one.
