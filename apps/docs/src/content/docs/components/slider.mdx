## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number[]` | — | Controlled. `[v]` = single; `[lo, hi]` = range. |
| `defaultValue` | `number[]` | `[0]` | Uncontrolled initial value(s). |
| `onValueChange` | `(values: number[]) => void` | — | Change handler. |
| `min` | `number` | `0` | Minimum value. |
| `max` | `number` | `100` | Maximum value. |
| `step` | `number` | `1` | Increment. |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Axis. |
| `tone` | `'primary' \| 'danger'` | `'primary'` | Range and thumb border colour. |
| `disabled` | `boolean` | `false` | Disables interaction. |

## Don't

- Don't ship a slider without a visible value readout — the user needs to know what they chose.
- Don't use `step` smaller than what the value reasonably represents (1 cent for currency, 1 px for sizes).
