## Quick start

```bash
pnpm add @talon-ui/react @talon-ui/tokens
```

### Tailwind project (recommended)

```ts
// tailwind.config.ts
import preset from '@talon-ui/tokens/preset';
export default {
  presets: [preset],
  content: [
    './src/**/*.{ts,tsx}',
    './node_modules/@talon-ui/react/dist/**/*.js',
  ],
  darkMode: ['class', '[data-theme="dark"]'],
};
```

```ts
// app entry
import '@talon-ui/tokens/css';
```

### Non-Tailwind project

```ts
import '@talon-ui/react/styles.css';
```

A single import gives you tokens (light + dark) plus every utility the components use.

## Design language

Tokens follow a three-layer model: primitive → semantic → component. Components consume only the semantic layer, so light / dark / future white-label themes flip via CSS variables — no `dark:` Tailwind variants anywhere in component source. See [Tokens overview](/tokens/overview/) and [Theming](/getting-started/theming/).
