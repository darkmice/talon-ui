# Switch

Binary on/off toggle on a track-and-thumb anatomy. `Switch` is the on/off toggle. Wraps `@radix-ui/react-switch` for accessibility and keyboard support.

## Don't

- Don't use `Switch` for choosing between multiple options — use `RadioGroup`.
- Don't combine with destructive labels without explicit confirmation. Toggles imply instant effect.
